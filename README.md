## girlcelly-magnet-extract

Simple script for grabbing magnet URLs from girlcelly.blog

### Usage

```
$ java -jar girlcelly-magnet-extract.jar <archive> <last-page>
```

e.g.
```
$ java -jar girlcelly-magnet-extract.jar blog-date-201704 4
```
